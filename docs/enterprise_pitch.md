# 石化行业大脑 — 企业价值方案

## 1. 产品定位

基于**「核心运行 + 底座支撑 + 自我进化」**三层架构的石化智能底座，为企业提供从预测维护、工艺寻优到催化剂研发的全场景 AI 能力，并具备持续从生产数据中学习、迭代进化的核心竞争力。

---

## 2. 核心差异化

### 自我进化引擎 vs 竞品对比

| 能力维度 | 中控 TPT2 | 中石化长城大模型 | 行业大脑 |
|----------|-----------|-----------------|----------|
| 推理能力 | ✓ 强 | ✓ 强 | ✓ 强 |
| 知识库检索 | ✓ | ✓ | ✓ |
| 持续学习 | ✗ 仅推理 | ✗ 仅推理 | ✓ 持续进化 |
| 离线进化 | ✗ | ✗ | ✓ 闭环迭代 |
| 在线微调 | ✗ | 部分 | ✓ 安全门控 |
| 多 Agent 协同 | 部分 | 部分 | ✓ 场景化编排 |

**核心价值**：竞品聚焦「推理」，我们聚焦「推理 + 持续进化」。模型可随企业生产数据、反馈闭环不断自我优化，实现从「一次训练」到「持续进化」的跃迁。

---

## 3. 三大典型场景与 ROI

### 场景一：设备预测维护

- **价值**：降低非计划停车 40%+，减少突发故障损失
- **参考案例**：镇海炼化 52 万台设备智能预警体系
- **ROI**：单次非计划停车损失百万级，年节约运维成本可观

### 场景二：工艺参数寻优

- **价值**：收率提升 2%+，能耗降低 8%+
- **参考案例**：大连西太年创效 6855 万元
- **ROI**：收率每提升 1% 对千万吨级炼厂效益显著

### 场景三：催化剂研发助手

- **价值**：研发周期缩短 30%，知识复用率提升 50%
- **ROI**：加速新品上市，降低研发试错成本

## 4. 部署方案

- **私有化部署**：全部组件部署于客户机房或专有云
- **开箱即用**：预置石化领域知识图谱、Agent 模板、典型场景配置
- **数据不出企业**：原始数据、模型、日志均保留在企业内网

---

## 5. 合作模式

```
POC 验证（4 周） → 试点部署（3 个月） → 全面推广
```

1. **POC 验证**：选择 1–2 个典型场景，在客户真实环境验证效果与可行性
2. **试点部署**：选定 1–2 套装置，接入生产数据，形成可复制的部署与运维方案
3. **全面推广**：按装置、工厂逐步推广，建立企业级智能底座
