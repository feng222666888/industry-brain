# 评估指标定义

本文档定义设备预测性维护场景的关键评估指标。

## 一、核心指标

### 1.1 预测准确率（Prediction Accuracy）

**定义**：故障预测的准确度，衡量系统正确识别故障的能力。

**计算公式**：
```
准确率 = (真阳性 + 真阴性) / (真阳性 + 假阳性 + 真阴性 + 假阴性)
```

**目标值**：>85%

**实际案例**：催化裂化装置轴承温度预警准确率92%

**评价维度**：
- 真阳性率（TPR）：正确识别故障的比例
- 假阳性率（FPR）：误报故障的比例
- 需要平衡TPR和FPR，避免过度预警或漏报

### 1.2 SOP完整率（SOP Completeness）

**定义**：生成的维修方案完整性，确保包含所有必要信息。

**评价维度**：
- 安全措施：是否包含完整的安全注意事项
- 工具清单：是否列出所有必需工具
- 备件清单：是否列出所有必需备件
- 步骤完整性：维修步骤是否完整、可执行

**目标值**：100%

**检查项**：
- [ ] 安全措施完整
- [ ] 工具清单完整
- [ ] 备件清单完整
- [ ] 步骤描述清晰
- [ ] 时间估算合理

### 1.3 端到端延迟（End-to-End Latency）

**定义**：从告警触发到生成完整SOP的时间。

**测量点**：
1. 告警触发时间
2. Monitor Agent完成时间
3. Diagnosis Agent完成时间
4. Repair Agent完成时间
5. SOP生成完成时间

**目标值**：<10秒

**分解目标**：
- Monitor Agent：<2秒
- Diagnosis Agent：<3秒
- Repair Agent：<5秒

## 二、经济效益指标

### 2.1 维护成本降低率

**定义**：实施预测性维护后，维护成本降低的百分比。

**计算公式**：
```
维护成本降低率 = (原维护成本 - 新维护成本) / 原维护成本 × 100%
```

**目标值**：22-28%

**实际案例**：某炼化企业维护成本降28%

### 2.2 非计划停机减少率

**定义**：实施预测性维护后，非计划停机次数减少的百分比。

**计算公式**：
```
非计划停机减少率 = (原非计划停机次数 - 新非计划停机次数) / 原非计划停机次数 × 100%
```

**目标值**：65%

**实际案例**：某炼化企业非计划停机减少65%

### 2.3 故障处理时间缩短

**定义**：从故障发生到处理完成的时间缩短。

**目标值**：从4.2小时缩至1.5小时

**实际案例**：某企业故障处理时间从4.2小时缩至1.5小时

### 2.4 设备寿命延长

**定义**：通过预测性维护，设备运行周期延长的百分比。

**实际案例**：
- 装置运行周期从18个月延长至36个月
- 换热器清洗周期从3个月延长至6个月

## 三、技术性能指标

### 3.1 设备平均无故障运行时间（MTBF）

**定义**：设备平均无故障运行时间。

**目标值**：提升130%

**实际案例**：某企业MTBF提升130%

### 3.2 重大故障率下降

**定义**：重大故障发生的频率下降。

**目标值**：下降55%

**实际案例**：某企业重大故障率下降55%

### 3.3 预测提前时间

**定义**：故障发生前能够提前预测的天数。

**目标值**：14天

**实际案例**：基于振动频谱分析，提前14天预测磨损趋势

## 四、数据质量指标

### 4.1 数据完整性

**定义**：传感器数据的完整程度。

**检查项**：
- 数据采集率：>99%
- 数据缺失率：<1%
- 数据异常率：<0.1%

### 4.2 数据时效性

**定义**：数据从采集到展示的延迟。

**目标值**：<200ms

**实际案例**：128项关键参数实时展示，延迟≤200ms

## 五、用户满意度指标

### 5.1 系统可用性

**定义**：系统正常运行的时间比例。

**目标值**：>99.5%

### 5.2 用户操作便捷性

**定义**：用户完成关键操作所需的步骤数。

**目标**：
- 设备选择：1步
- 触发诊断：1步
- 查看SOP：1步

## 六、ROI指标

### 6.1 投资回报率

**计算公式**：
```
ROI = (收益 - 成本) / 成本 × 100%
```

**实际案例**：
- 年节约成本：4500万元
- 节电量：43万千瓦时
- 节约电费：12.1万元

### 6.2 成本节约

**实际案例**：
- 维护成本降低：28%
- 备件资金占用下降：45%
- 库存周转天数从98天降至56天

## 七、指标监控

### 7.1 实时监控

- 健康评分：实时更新
- 传感器数据：实时流式传输
- Agent执行状态：实时展示

### 7.2 历史分析

- 健康趋势：过去30天
- 故障历史：过去1年
- 维修记录：过去1年

### 7.3 报表生成

- 日报：每日健康状态汇总
- 周报：周度故障分析
- 月报：月度维护成本分析
