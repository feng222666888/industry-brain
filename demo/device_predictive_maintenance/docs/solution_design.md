# 设备预测性维护落地方案设计

## 一、技术架构

### 1.1 三层架构

基于现有三层架构，通过Multi-Agent协同实现：

```
传感器数据 → Monitor Agent（异常检测）
           ↓
    Diagnosis Agent（根因分析）
           ↓
    Repair Agent（SOP生成）
           ↓
    维修方案输出
```

### 1.2 核心能力展示点

1. **实时监控与预测**：基于振动、温度等传感器数据的实时健康评分
2. **智能诊断**：Multi-Agent协同，从异常检测到根因分析
3. **自动SOP生成**：基于知识库自动生成标准化维修方案
4. **可视化展示**：设备拓扑、健康趋势、诊断流程可视化

## 二、演示流程设计

### 2.1 用户交互流程

1. **设备选择**：展示多个关键设备及其健康状态
2. **实时监控**：展示传感器数据流和健康趋势
3. **触发诊断**：模拟异常告警，触发Multi-Agent诊断流程
4. **结果展示**：展示诊断结果、根因分析、维修SOP

### 2.2 技术实现

#### 数据层
- 设备元数据：10个不同类型的设备
- 传感器时序数据：30天的历史数据
- 故障历史：20条历史故障记录
- 维修记录：30条维修记录
- 知识库：故障模式、SOP模板

#### API层
- `/api/device-demo/devices` - 设备列表
- `/api/device-demo/devices/{id}/health` - 健康状态
- `/api/device-demo/devices/{id}/sensor-data` - 传感器数据
- `/api/device-demo/devices/{id}/diagnose` - 诊断触发（SSE流式）

#### 前端层
- 健康仪表盘：实时健康评分和趋势
- 3D设备视图：设备状态可视化
- 实时数据流：传感器数据波形图
- Agent流程可视化：Multi-Agent协作流程
- SOP卡片：维修方案展示

## 三、核心功能模块

### 3.1 设备实时监测面板

- 128项关键参数实时展示，延迟≤200ms
- 三级预警阈值可视化（催化裂化装置轴承温度预警准确率92%）

### 3.2 预测性维护引擎

- 基于振动频谱分析的故障预测（提前14天预测）
- 备件库存智能预测（准确率85%）

### 3.3 数字孪生模型

- 设备状态实时映射
- 健康状态可视化

### 3.4 故障知识图谱

- 整合2000+历史故障记录与解决方案
- 3秒内推送诊断建议与验证步骤

## 四、技术栈

### 4.1 前端技术

- **Next.js 15**：React框架
- **Tailwind CSS**：现代化样式
- **ECharts**：数据可视化
- **ReactFlow**：Agent流程可视化
- **TypeScript**：类型安全

### 4.2 后端技术

- **FastAPI**：Python Web框架
- **Multi-Agent Engine**：现有Agent编排引擎
- **SSE**：流式数据传输

### 4.3 数据生成

- **NumPy/SciPy**：科学计算
- **物理模型**：基于故障特征频率的振动信号生成
- **Weibull分布**：设备退化曲线模拟

## 五、演示数据策略

### 5.1 数据范围

- **多设备类型**：离心泵、压缩机、换热器、反应器等（10+设备）
- **多故障类型**：轴承故障（内圈/外圈/滚动体）、密封失效、气蚀、不平衡等
- **完整时间序列**：过去30天的传感器数据（振动、温度、压力、流量等）
- **历史记录**：故障历史、维修记录、备件消耗记录

### 5.2 数据生成方法

- 使用物理模型生成真实的振动信号（包含故障特征频率）
- 基于Weibull分布生成设备退化曲线
- 模拟真实的生产环境波动（温度、负载变化）

## 六、UI设计理念

### 6.1 设计风格

参考Google/Meta的现代科技感设计：
- **深色主题**：深蓝/黑色背景，高对比度
- **动态效果**：数据流动画、粒子效果、渐变光效
- **3D可视化**：设备3D模型、拓扑图
- **实时反馈**：SSE流式更新、进度指示、状态动画

### 6.2 视觉特点

- 极简布局 + 彩色高亮数据
- 大屏友好的数据可视化
- 实时动态图表与热力图
- 深色主题 + 霓虹配色方案

### 6.3 交互特点

- 一键调用预测模型
- 拖拽式故障诊断面板
- AR/3D设备模型可视化
- 自适应响应式设计
